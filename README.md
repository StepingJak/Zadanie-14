Aby uruchomić DataTable należy załadować bibliotekę tagiem script z źródłem: <script src="https://cdn.datatables.net/2.1.8/js/dataTables.js"></script>.  
Następnie należy stworzyć tabelę w HTML z tagiem <thead> definicującym kolumny.  
Finalnie w skrypcie definiujemy stałą której wartościa jest instancja tabeli w HTML i definiujemy wszelakie właściwości (kolumny, przyciski)  
W aplikacji wykorzystane zostały funkcje:  
 -.on("click" "przycisk", funkcja)  
 -prompt("wiadomość", zmienna)
 -.draw()
 -.row, .parents, ,.data
